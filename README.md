## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository with advanced authentication features including Google OAuth.

## Features

- **Local Authentication**: Email/phone registration and login with verification
- **Google OAuth**: Seamless Google authentication and registration
- **JWT Tokens**: Access and refresh token management
- **Email/SMS Verification**: Multi-channel verification system
- **User Management**: Complete user CRUD operations

## Environment Variables

Create a `.env` file in the root directory with the following variables:

```env
# Database
DATABASE_URL="postgresql://username:password@localhost:5432/react-course?schema=public"

# JWT Configuration
JWT_SECRET="your-super-secret-jwt-key-here"
JWT_ACCESS_EXPIRATION_TIME="15m"
JWT_REFRESH_EXPIRATION_TIME="7d"

# Google OAuth Configuration
GOOGLE_CLIENT_ID="your-google-client-id"
GOOGLE_CLIENT_SECRET="your-google-client-secret"
GOOGLE_CALLBACK_URL="http://localhost:3000/auth/google/callback"

# Frontend URL (for OAuth redirects)
FRONTEND_URL="http://localhost:3001"

# Email Configuration (for nodemailer)
EMAIL_HOST="smtp.gmail.com"
EMAIL_PORT=587
EMAIL_USER="your-email@gmail.com"
EMAIL_PASS="your-app-password"

# SMS Configuration (for Vonage)
VONAGE_API_KEY="your-vonage-api-key"
VONAGE_API_SECRET="your-vonage-api-secret"
VONAGE_FROM_NUMBER="your-vonage-phone-number"
```

## Google OAuth Setup

1. Go to the [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project or select an existing one
3. Enable the Google+ API
4. Go to "Credentials" and create an OAuth 2.0 Client ID
5. Add your redirect URI: `http://localhost:3000/auth/google/callback`
6. Copy the Client ID and Client Secret to your `.env` file

## API Endpoints

### Authentication

- `POST /auth/register` - Register with email/phone
- `POST /auth/login` - Login with email/phone and password
- `POST /auth/verify` - Verify registration code
- `POST /auth/resend-code` - Resend verification code
- `POST /auth/refresh` - Refresh access token

### Google OAuth

- `GET /auth/google` - Initiate Google OAuth flow
- `GET /auth/google/callback` - Google OAuth callback
- `POST /auth/google` - Direct Google authentication (for mobile apps)

### Users

- `GET /users/profile` - Get user profile (protected)
- `PUT /users/profile` - Update user profile (protected)
